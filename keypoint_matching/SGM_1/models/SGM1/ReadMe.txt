This folder contains the code for StableGM-v1 model.

For StableGM-v1, we have adapted the neural architecture and affinity layer proposed in the paper - Michal Rolínek, Paul Swoboda, Dominik Zietlow, Anselm Paulus, Vít Musil, Georg Martius. "Deep Graph Matching via Blackbox Differentiation of Combinatorial Solvers." ECCV 2020 and it's implementation available in Thinkmatch - https://github.com/Thinklab-SJTU/ThinkMatch/.
